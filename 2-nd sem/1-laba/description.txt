Алгоритм Грэхема (Graham's scan) — это эффективный алгоритм для построения выпуклой оболочки множества точек на плоскости. Выпуклая оболочка — это минимальный выпуклый многоугольник, содержащий все точки множества. Алгоритм работает за время O(n log n), где n — количество точек.

Основные шаги алгоритма Грэхема:
Нахождение стартовой точки:

Находим точку с минимальной координатой y (самую нижнюю точку). Если таких точек несколько, выбираем самую левую из них. Эта точка гарантированно будет входить в выпуклую оболочку.

Сортировка точек по полярному углу:

Все остальные точки сортируются по возрастанию полярного угла относительно стартовой точки. Полярный угол вычисляется с помощью функции atan2(dy, dx), где dy и dx — разности координат текущей точки и стартовой точки.

Если несколько точек имеют одинаковый полярный угол, оставляем только самую дальнюю от стартовой точки (остальные удаляем, так как они не могут быть частью выпуклой оболочки).

Построение выпуклой оболочки:

Используем стек для хранения точек, которые могут быть частью выпуклой оболочки.

Проходим по отсортированным точкам и для каждой точки проверяем, образует ли она выпуклый угол с последними двумя точками в стеке. Это делается с помощью векторного произведения (cross product).

Если текущая точка образует невыпуклый угол (поворот по часовой стрелке), удаляем последнюю точку из стека.

Повторяем этот процесс, пока все точки не будут обработаны.
